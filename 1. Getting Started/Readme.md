## Follow these steps (after setting up your OAK):

1. Open your python CMD environment (Anaconda recommended)
2. Run:

 ```python
 conda install git
 git clone https://github.com/luxonis/depthai.git
 cd dephtai
 python install_requirements.py
 python depthai_demo.py
 ```
 
 3. Done!
